## 🚀 Краткое описание

ChangelogAI — плагин для JetBrains IDE и GigaIDE, который **автоматически формирует CHANGELOG.md**
на основе git-коммитов, используя GigaChat для классификации и описания изменений.

## ✨ Особенности

- **💬 AI-Чат с агентным оркестратором (GigaCodeAE)**: Архитектура «оркестратор + суб-агенты» — автоматическая классификация задач, чтение/запись файлов, выполнение команд в терминале, поиск по коду. Суб-агенты: Code, Review, Test, Search, Planner, Summarizer, Tool. Система скиллов — кастомизируемые системные промпты для разных режимов работы
- **⚡ Sprint Analyzer — War Room Dashboard**: AI-анализ активного Jira-спринта через MCP: автоматическое выявление рисков, блокеров и неоднозначных задач. Декомпозиция историй на подзадачи и их создание в Jira одним кликом
- **📝 Spec Generator**: Многоагентная генерация полных технических требований (FR, NFR, AC, Edge Cases), PRD и Mermaid-диаграмм. Загрузка требований из Confluence, оценка качества существующего ТЗ, оценка трудозатрат и **автоматическое создание Jira Story + подзадач** из результатов оценки
- **🔍 Jenkins Dashboard**: AI-анализ пайплайнов и сборок прямо в IDE: мониторинг статусов, диагностика причин падений, интерактивный чат по логам
- **📚 База знаний (Knowledge Base)**: Индексирование страниц Confluence с векторными эмбеддингами. Автоматический RAG-контекст в чате и генераторе спецификаций (BM25 + векторный поиск). Доступна как MCP-инструмент `search_knowledge_base`
- **🔗 MCP (Model Context Protocol)**: Настройка через `.mcp.json` в стиле Claude Code / Cursor. Поддержка STDIO и HTTP-транспорта. Авто-обнаружение серверов из Cursor, Claude Desktop, глобального конфига
- **🤖 Changelog Generator**: Выбираете git-коммиты — плагин группирует их по типам и формирует CHANGELOG.md. Защита от дублей, кастомный промпт
- **📊 Test Coverage**: Аналитика покрытия кода на базе JaCoCo XML: прогресс-бары по классам/методам/инструкциям, таблица непокрытых методов
- **🔌 Гибкое подключение**: Внутренний GigaChat (mTLS), внешний GigaChat (OAuth), Chemoda и любой OpenAI-совместимый API. Отдельная настройка Embedding API

## 📦 Установка

### Ручная установка

1. Скачайте последнюю версию плагина для вашей версии IDE
2. Перейдите в Settings (Ctrl+Alt+S) → Plugins → **«⚙»** → «Install Plugin from Disk...» → выберите файл
3. Перезапустите IDE
4. Откройте тулвиндоу **ChangelogAI** в правой панели
5. На вкладке **Settings** введите:
   - **AI token** — токен Чемодана или внешнего GigaChat
     **↑ ИЛИ ↓**
   - Пути к **сертификату** и **приватному ключу** в полях AI cert path / AI key path
6. Нажмите **Apply**

## 🛠 Использование

1. Откройте тулвиндоу **ChangelogAI** в правой панели IDE
2. Перейдите на вкладку **Changelog Generator**
3. Выберите коммиты из списка (по умолчанию последние 50)
4. Нажмите **Generate** — ИИ сформирует changelog в области Preview
5. Сохраните результат в `CHANGELOG.md` или скопируйте в буфер обмена
6. Для кастомизации промпта нажмите кнопку **Edit** (иконка карандаша)
