## 🚀 Краткое описание

ChangelogAI — плагин для JetBrains IDE и GigaIDE, который **автоматически формирует CHANGELOG.md**
на основе git-коммитов, используя GigaChat для классификации и описания изменений.

## ✨ Особенности

- **🤖 AI-генерация changelog**: Выбираете коммиты — плагин группирует их по типам (Added, Fixed, Changed, Removed) и формирует Markdown
- **🔁 Защита от дублей**: Коммиты, уже попавшие в CHANGELOG.md, автоматически пропускаются при повторной генерации
- **✏️ Кастомный промпт**: Редактируйте промпт прямо в IDE или через файл `.changelogai/prompt_ru.md` в корне проекта
- **🔌 Гибкое подключение**: Поддержка внутреннего GigaChat (mTLS), внешнего GigaChat (OAuth) и Chemoda
- **🎛 Фича-тоглы**: Включайте и отключайте функции (Changelog, AI Chat, Spec Generator, MCP) без перезапуска IDE
- **💬 Умный AI-Чат**: Встроенный ассистент для написания и рефакторинга кода
- **🔗 Поддержка MCP (Model Context Protocol)**: Подключайте локальные инструменты и сервисы (в т.ч. Atlassian/Jira) напрямую. Включает удобный инспектор JSON-запросов к инструментам (Cursor-style)
- **📝 Spec Generator**: Автоматическое создание подробных технических заданий и Use Cases на лету
- **📊 Анализ Coverage**: Интеллектуальная аналитика покрытия вашего кода тестами прямо в IDE

## 📦 Установка

### Ручная установка

1. Скачайте последнюю версию плагина для вашей версии IDE
2. Перейдите в Settings (Ctrl+Alt+S) → Plugins → **«⚙»** → «Install Plugin from Disk...» → выберите файл
3. Перезапустите IDE
4. Откройте тулвиндоу **ChangelogAI** в правой панели
5. На вкладке **Settings** введите:
   - **AI token** — токен Чемодана или внешнего GigaChat
     **↑ ИЛИ ↓**
   - Пути к **сертификату** и **приватному ключу** в полях AI cert path / AI key path
6. Нажмите **Apply**

## 🛠 Использование

1. Откройте тулвиндоу **ChangelogAI** в правой панели IDE
2. Перейдите на вкладку **Changelog Generator**
3. Выберите коммиты из списка (по умолчанию последние 50)
4. Нажмите **Generate** — ИИ сформирует changelog в области Preview
5. Сохраните результат в `CHANGELOG.md` или скопируйте в буфер обмена
6. Для кастомизации промпта нажмите кнопку **Edit** (иконка карандаша)
