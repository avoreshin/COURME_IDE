GIGACOURME

AI-ассистент для разработчиков: агентный чат, Jira-спринты, технические спецификации, база знаний и MCP-инструменты — всё в одном тулвиндоу.

🚀 Краткое описание

GIGACOURME — плагин для JetBrains IDE и GigaIDE, который объединяет несколько AI-инструментов в удобном боковом тулвиндоу с навигацией по иконкам. Подключается к GigaChat (корпоративному или внешнему), а также к любому OpenAI-совместимому API.

✨ Возможности

  • 💬 AI-Чат с агентным оркестратором (GigaCodeAE): Архитектура «оркестратор + суб-агенты» — автоматическая классификация задач, чтение/запись файлов, выполнение команд в терминале, поиск по коду. Суб-агенты: Code, Review, Test, Search, Planner, Summarizer, Tool (MCP). Система скиллов — кастомизируемые системные промпты для разных режимов работы
  • ⚡ Sprint Analyzer — War Room Dashboard: AI-анализ активного Jira-спринта через MCP: автоматическое выявление рисков, блокеров и неоднозначных задач. Интерактивный дашборд на JCEF с декомпозицией историй на подзадачи и их созданием в Jira одним кликом
  • 📝 Spec Generator: Многоагентная генерация полных технических требований (FR, NFR, AC, Edge Cases), PRD-документа и Mermaid-диаграмм из краткого описания задачи. Поддерживает загрузку требований из Confluence, оценку качества существующего ТЗ и оценку трудозатрат
  • 📚 База знаний (Knowledge Base): Загружает страницы из Confluence, разбивает на чанки, генерирует векторные эмбеддинги и сохраняет локально. При генерации требований и в чате автоматически подтягивает релевантный контекст через RAG (BM25 + векторный поиск). Доступна как MCP-инструмент search_knowledge_base
  • 🔗 MCP (Model Context Protocol): Настройка через JSON-файл .mcp.json в стиле Claude Code / Cursor. Поддержка STDIO и HTTP-транспорта. Авто-обнаружение серверов из Cursor, Claude Desktop, глобального конфига. Тест подключений с выводом списка инструментов
  • 🤖 Changelog Generator: Выбираете git-коммиты — плагин группирует их по типам (Added, Fixed, Changed, Removed) и формирует CHANGELOG.md
  • 📊 Test Coverage: Аналитика покрытия кода юнит-тестами на базе JaCoCo XML: прогресс-бары по классам/методам/инструкциям, таблица непокрытых методов
  • 🔌 Гибкое подключение к LLM: Внутренний GigaChat (mTLS), внешний GigaChat (OAuth), Chemoda и любой OpenAI-совместимый API. Отдельная настройка Embedding API

🛠 Быстрый старт

  1. Откройте тулвиндоу GIGACOURME в правой панели IDE
  2. Настройте подключение через иконку ⚙ Settings (URL, токен или сертификат)
  3. MCP-серверы: отредактируйте .mcp.json в корне проекта или через иконку 🔌 MCP Servers
  4. База знаний: укажите Embedding URL в Settings, затем проиндексируйте Confluence на вкладке Knowledge Base
  5. Sprint Analyzer: подключите Jira через MCP, откройте вкладку Sprint Analyzer и нажмите Analyze Sprint
