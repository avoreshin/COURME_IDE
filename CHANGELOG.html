v1.0.0 — первый релиз

  • Агентный AI-чат GigaCodeAE: оркестратор + суб-агенты (Code, Review, Test, Search, Planner, Summarizer, Tool), скиллы, @-упоминания файлов
  • Sprint Analyzer — War Room Dashboard на JCEF: AI-анализ Jira-спринта через MCP, декомпозиция историй, создание подзадач в Jira
  • Spec Generator: многоагентная генерация FR/NFR/AC/Edge Cases/PRD/Mermaid из описания задачи; загрузка требований из Confluence; оценка качества ТЗ и трудозатрат
  • Knowledge Base: RAG из Confluence — эмбеддинги + BM25-индекс, семантический поиск, MCP-инструмент search_knowledge_base
  • MCP: настройка через .mcp.json (совместим с Claude Code / Cursor), STDIO и HTTP транспорт, авто-обнаружение конфигов, тест подключений
  • Changelog Generator: выбор git-коммитов → группировка по типам → CHANGELOG.md
  • Test Coverage: парсинг JaCoCo XML, прогресс-бары, таблица непокрытых методов
  • Поддержка GigaChat (mTLS / OAuth), Chemoda, OpenAI-совместимых API, отдельный Embedding API
  • Навигация по иконкам в стиле VS Code — Settings вынесены в отдельную иконку внизу
  • Фича-тоглы — включение/отключение модулей без перезапуска IDE
